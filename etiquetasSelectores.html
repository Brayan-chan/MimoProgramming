Para hacer que las páginas web más grandes sean más manejables, podemos mover nuestro CSS a una hoja de estilo o a un archivo especial solo para diseñar la página web.

CSS 

        body {
        font-family: "Helvetica Neue";
        text-align: center;
        }
            
        h1 {
        color: darkBlue;
        }
            
        img {
        border-radius: 10px;
        margin-top: 8px;
        margin-left: 2px;
        margin-right: 2px;
        }

Para incluir una hoja de estilo en un archivo HTML, usamos el elemento link.Es un elemento vacío de cierre automático y va dentro del elemento head.

HTML 

    <!doctype html>
    <html>
    <head>
    <link>
        <meta charset="utf-8">
        <title>Selectores</title>
        <style>
            body {
                font-family: "Helvetica Neue";
                text-align: center;
            }
            
            h1 {
                color: darkblue;
            }
            
            img {
                border-radius: 10px;
                margin-top: 8px;
                margin-left: 2px;
                margin-right: 2px;
            }
        </style>
        <html/>

Para saber qué tipo de archivo incluir, la etiqueta de link de apertura necesita el conjunto de atributo rel usando rel="stylesheet".

HTML 

    <!doctype html>
    <html>
    <head>
        <link rel = "stylesheet">
    </head>
    <body>
        <h1>Selectores</h1>
        <p>Why are rivers amazing roommates?
        They go with the flow.</p>
    </body>
    </html>

Para especificar la ubicación de la hoja de estilo, establezca el atributo href en "style.css".

HTML 

    <!doctype html>
    <html>
    <head>
        <link rel = "stylesheet" href="style.css">
    </head>
    <body>
        <h1>Selectores</h1>
        <p>Why are rivers amazing roommates?
        They go with the flow.</p>
    </body>
    </html>

CONCLUSIONES:

"stylesheet" es el tipo de archivo que se va insertar y href es el atributo que ayuda al elemento de link a encontrar la hoja de estilo.
