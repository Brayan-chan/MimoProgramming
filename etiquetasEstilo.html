Uso de etiquetas de estilo.

¡Las páginas web modernas tienen mucho estilo!. Eso significa que tienen muchas propiedades para que se vean geniales.

Si queremos que un estilo se aplique a toda la página, podemos utilizar el selector de body.

CSS 

            body {
                background-color: lightblue;
            }

Establecer el color de fondo en lightblue; dentro de la regla del body establece el fondo para el cuerpo.

Estiilos como color: black; cuando se establecen dentro de la regla del body, se aplicará a todos los elementos dentro del body.

CSS 

                body {
                    background-color: lightblue;
                    color: black;
                }

Podemos diseñar tantos elementos como queramos dentro de las etiquetas de style, siempre que las agreguemos un espacio entre las diferentes reglas.
