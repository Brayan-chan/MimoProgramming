CSS BÁSICO

Ahora pasaremos a crear sitios web atractivos con la ayuda de un lenguaje de estilo llamado CSS, lo cuál es la abreviatura de hojas de estilo en cascada.

Practiquemos cambiando un elemento de encabezado a blue.

html 

            <h2 style="color: blue;">Delightful Flowers by Brayan</h2>

Delightful Flowers by Brayan

Para agregar estilo a un documento HTML, comenzamos con los atributos style.

html 

            <h1 style>Agregando el atributo style</h1>

Dado que style es un atributo, necesitamos el signo =, seguido de comillas "" para darle un valor.

html 

            <h1 style="">Agregando los signos del atributo style</h1>

Cuando agregamos algo como color: green;, estamos agregando una declaración. Las declaraciones son CSS, no HTML

Una declaración se hace a partir de una propiedad CSS, como color y un valor, como pink. La propiedad siempre es lo primero.

Agregamos : entre la propiedad y el valor, y ; al final de la declaración.

Hay muchas propiedades CSS. Por ejemplo para cambiar el color de un elemento, usamos background-color y lo establecemos en un valor como darkGreen.

CSS 

            <h1 style="background-color: darkGreen">Agregando el atributo style</h1>

Agregando el atributo style

Podemos agregar tantas propiedades como queramos siempre que terminemos cada una con un punto y coma.

CSS 

            <h1 style="color: pink; background-color: darkGreen">Agregando el atributo style</h1>

Agregando el atributo style
